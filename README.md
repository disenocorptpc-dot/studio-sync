# Studio Sync | Diseño Industrial Dashboard

Dashboard de seguimiento de producción para equipos de diseño industrial y modelado 3D.

## Características
- **Vista de Tarjetas**: Estado actual de cada diseñador (Bocetaje, Modelado, Render).
- **Semáforo de Entregas**: Alertas visuales para proyectos urgentes (< 48h) o retrasados.
- **Bitácora Integrada**: Espacio para notas rápidas de avance.
- **Modo Oscuro Premium**: Interfaz optimizada para largas sesiones de trabajo.

## Instalación y Despliegue

### Local
1. Abrir `index.html` en tu navegador.
2. Los datos se guardan automáticamente en tu navegador (`localStorage`).

### Migración a GitHub
1. Crear un nuevo repositorio en GitHub.
2. Ejecutar:
   ```bash
   git remote add origin https://github.com/TU_USUARIO/studio-sync.git
   git branch -M main
   git push -u origin main
   ```

### Despliegue en Vercel
1. Ir a [Vercel](https://vercel.com).
2. Importar el repositorio de GitHub.
3. Deploy (No requiere confirguración adicional).
